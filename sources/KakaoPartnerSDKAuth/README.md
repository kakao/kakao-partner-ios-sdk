# KakaoPartnerSDKAuth

파트너용 사용자 인증 및 토큰 관리 모듈입니다. `KakaoSDKAuth` 모듈에 의존합니다.

## Requirements
- Xcode 11.0
- iOS 11.0
- Swift 5.0
- CocoaPods 1.8.0

## Dependencies
- KakaoSDKAuth
- KakaoPartnerSDKCommon

## Installation
```
pod 'KakaoPartnerSDKAuth'
```

## Import
```
import KakaoPartnerSDKAuth
```
