# KakaoPartnerSDKLink

파트너용 카카오링크 모듈입니다. `KakaoSDKLink` 모듈에 의존합니다.

## Requirements
- Xcode 11.0
- iOS 11.0
- Swift 5.0
- CocoaPods 1.8.0

## Dependencies
- KakaoSDKLink
- KakaoPartnerSDKCommon

## Installation
```
pod 'KakaoPartnerSDKLink'
```

## Import
```
import KakaoPartnerSDKLink
```

## Usage
