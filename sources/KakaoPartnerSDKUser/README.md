# KakaoPartnerSDKUser

파트너용 카카오 로그인 및 사용자 관리 API 모듈입니다. 로그인 기반 API를 제공하므로 `KakaoPartnerSDKAuth` 모듈에 의존합니다.

## Requirements
- Xcode 11.0
- iOS 11.0
- Swift 5.0
- CocoaPods 1.8.0

## Dependencies
- KakaoSDKUser
- KakaoPartnerSDKAuth

## Installation
```
pod 'KakaoPartnerSDKUser'
```

## Import
```
import KakaoPartnerSDKUser
```
